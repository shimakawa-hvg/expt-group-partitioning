# expt-group-partitioning
"Prevention of Leakage in Machine Learning Prediction for Polymer Composite Properties" 

Hajime Shimakawa, Akiko Kumada, and Masahiro Sato

Department of Electrical Engineering & Information Systems, School of Engineering, 
The University of Tokyo, 7-3-1 Hongo, Bunkyo-ku, 113-8656, Tokyo, Japan

## Requirements
- python 3.7
## Content
- ```data_EC.csv```: csv file for data used in EC prediction.
- ```data_TC.csv```: csv file for data used in TC prediction.
- ```make_expt_id.ipynb```: jupyternotebook for making expt-id from reference information.
- ```sample_detail_EC.csv```: csv file for sample details used to make expt-id.
- ```sample_detail_TC.csv```: csv file for sample details used to make expt-id.
## Reference
We used experimental data from PoLyInfo (https://polymer.nims.go.jp/) as ```sample_detail_XX.csv```.
